Django shortcuts
================

You spend way too much time typing `python manage.py [...]`.

Usage
-----
    
    $ django shell
    
    $ cd any/project/subdirectory
    $ django shell

Shortcuts
---------

* `r` = `runserver`
* `s` = `shell`
* `schemamigration` = `sm`
* `migrate` = `m`
* `test` = `t`
* `dbshell` = `d`
* `syncdb` = `sd`
* `startproject` = `sp`

Installation
------------

::

    $ pip install django-shortcuts
